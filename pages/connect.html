How to Connect

Get Online in Minutes

Connecting to Easy Link is Simple

Follow these easy steps to get online with Easy Link Wi-Fi. The process is quick and straightforward, requiring just a voucher code and a few clicks.

1. Purchase a Voucher

Buy an Easy Link voucher from any authorized reseller in Arua. Vouchers are available in various denominations for different time periods (7 hours, 24 hours, 7 days, etc.).

Look for shops and kiosks displaying the Easy Link logo. See our Coverage page for a list of authorized resellers.

2. Turn On Wi-Fi & Select Easy Link Network

Enable Wi-Fi on your device (phone, laptop, tablet) and look for the "Easy Link" network in your list of available networks.

Make sure you're within range of an Easy Link hotspot. The signal is strongest within 100 meters of a hotspot location.

3. Connect to the Network

Tap or click on the "Easy Link" network to connect. No password is required for the initial connection.

If the login page doesn't appear automatically, open your browser and try to visit any website. You should be redirected to the login page.

4. Enter Your Voucher Code

A login page will automatically appear. Enter your voucher code in the provided field and click "Connect" or "Login".

Enter the code exactly as it appears on your voucher, including any dashes or special characters. Codes are case-sensitive.

5. Start Browsing

Once your code is accepted, you'll be connected to the internet! You can now browse, stream, download, and enjoy all your online activities.

Your session will remain active for the duration of your package or until you disconnect. If you disconnect, you can reconnect using the same code until it expires.

Troubleshooting Connection Issues

Having trouble connecting? Here are solutions to common issues:

Login Page Not Appearing

  • Try opening your browser and visiting any website
  • Clear your browser cache and cookies
  • Try using a different browser
  • Disconnect and reconnect to the Wi-Fi network

Code Not Accepted

  • Double-check for typing errors
  • Ensure the code hasn't expired
  • Verify you're connected to an Easy Link network
  • Check if the code has already been used (for single-use vouchers)

Slow Connection

  • Move closer to the hotspot location
  • Check if many users are connected (peak hours)
  • Restart your device
  • Close bandwidth-intensive applications

Disconnected Unexpectedly

  • Check if your voucher has expired
  • Ensure you're still within range of the hotspot
  • Reconnect to the network and re-enter your code if needed
  • Restart your device if the problem persists
